# Финансовый менеджер

Простой консольный финансовый менеджер на чистом JavaScript для учета доходов и расходов.

## Описание

Это учебный проект, демонстрирующий базовый синтаксис JavaScript. Программа позволяет отслеживать баланс, добавлять транзакции, фильтровать их по различным критериям и анализировать финансовую активность.

## Основные функции

- **Учет баланса**: Отображение текущего баланса в выбранной валюте
- **Управление транзакциями**: Добавление новых транзакций (доходы/расходы)
- **Просмотр истории**: Отображение всех транзакций в удобном формате
- **Фильтрация**: Фильтрация транзакций по типу (доход/расход)
- **Категориальный анализ**: Подсчет общей суммы по категориям
- **Поиск**: Поиск транзакций по ключевым словам в описании
- **Проверка средств**: Проверка возможности совершить трату на основе текущего баланса

## Структура данных

### Транзакция
Каждая транзакция содержит:
- `id`: Уникальный идентификатор
- `type`: Тип операции ("доход" или "расход")
- `category`: Категория (еда, транспорт, зарплата и т.д.)
- `amount`: Сумма
- `date`: Дата в формате YYYY-MM-DD
- `description`: Описание транзакции

### Глобальные переменные
- `balance`: Текущий баланс (изначально 10000 RUB)
- `currency`: Валюта (по умолчанию "RUB")
- `transactions`: Массив всех транзакций

## Использование

### Функции

1. **showBalance()** - Показать текущий баланс
2. **addTransaction(type, category, amount, description)** - Добавить новую транзакцию
3. **showAllTransactions()** - Показать все транзакции
4. **filterTransactionsByType(type)** - Фильтровать транзакции по типу
5. **getTotalByCategory(category)** - Получить общую сумму по категории
6. **canAfford(amount)** - Проверить возможность совершить трату
7. **searchTransactions(keyword)** - Найти транзакции по ключевому слову
8. **runTests()** - Запустить демонстрацию работы всех функций

### Примеры использования

```javascript
// Показать баланс
showBalance();

// Добавить транзакцию
addTransaction("расход", "еда", 500, "Обед в кафе");

// Показать все транзакции
showAllTransactions();

// Фильтровать расходы
filterTransactionsByType("расход");

// Получить сумму по категории
getTotalByCategory("развлечения");

// Проверить возможность траты
canAfford(5000);

// Поиск транзакций
searchTransactions("зарплата");
```

## Запуск проекта

1. Скопируйте код в файл с расширением `.js` (например, `finance-manager.js`)
2. Запустите с помощью Node.js:
   ```bash
   node finance-manager.js
   ```
3. Или в браузере:
   - Создайте HTML-файл
   - Подключите JavaScript-файл
   - Откройте консоль разработчика в браузере

## Демонстрация

При запуске функции `runTests()` программа продемонстрирует все возможности:

1. Покажет начальный баланс
2. Добавит несколько тестовых транзакций
3. Отобразит все транзакции
4. Выполнит фильтрацию по типу
5. Посчитает сумму по категории
6. Выполнит поиск по ключевым словам
7. Проверит возможность совершения трат
